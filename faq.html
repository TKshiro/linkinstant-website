よくある質問 (FAQ)

LinkInstantに関するFAQや、AI名刺管理の活用方法についてお答えします。

Kintone以外のCRM（SalesforceやHubSpot）とも連携できますか？
現在は「Excel / CSV エクスポート」および「Kintone API 直連携」に標準対応しております。 エンタープライズ版のお客様向けには、個別開発でSalesforceやPipedrive等とのAPI連携設定も承っておりますので、お問い合わせください。
無料プランの制限について教えてください。

個人向けの無料プランでは、AI名刺データ化が月間30枚までに制限されています。また、CSV/Excelへの「データエクスポート」のほか、複数人での「チーム共有」や「Kintone自動出力」などの連携機能は、すべてプレミアムプラン（年額6,000円 または 月額500円）以上でのご提供となります。

電波のない展示会会場でも名刺スキャン（OCR）はできますか？
はい、可能です。LinkInstantのハイブリッドOCR機能なら、電波がない場所でも「ローカルスキャンキュー」に名刺画像を次々と保存できます。電波が回復したタイミングで、AIがバックグラウンドで全件一括データ化を行います。
企業で導入する際、セキュリティは大丈夫ですか？
データは全てSSL/TLSで暗号化され、世界最高水準の堅牢性を誇るGoogle Cloud上のデータベースに保存されます。また、管理者アカウントからは、いつ誰がどのデータを出力したかをトラッキングできる「監査ログ機能」も備えており、金融機関レベルのセキュリティ要件にも対応しています。
